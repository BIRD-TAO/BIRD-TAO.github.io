“隐含双射自编码器”读书笔记

发表于 2020-04-15 | 分类于 Deep Learning (Generative Models) | |

引言

因为个人原因，已经有很久没有更新博文了。然而在这停更的近两年中，小林同学还是在不断学习进步与成长，也对自己未来的学习目标更加明确了。

近期对于一篇2019年小米AI实验室产出的“对隐含双射自编码器进行对抗学习”(arXiv:1906.08090v1 [cs.LG] 19 Jun 2019)的论文进行了详细的阅读以及简化版本的编程实现。在此进行分享。

背景介绍：

生成模型目前作为深度学习领域中无监督学习类别中的一大热门，其核心思想为：将一个pixel x pixel 维的张量（图片）的概率分布进行参数/非参数的估计（训练过程），从而能够在估计的分布中进行抽样，生成与训练集类似风格的图片。确切来说，深度学习（基于神经网络）框架下的生成模型应为非参数模型，因为该模型并未对概率的分布（如正态分布，指数分布）进行提前的给定。

然而，事实上对于成千上万张图片而言，每张图片的像素点数目庞大（如128x128的图片，其像素点高达上万），并且相邻的像素点直接实际上都是强相关的，若用$\textbf{X} = (X_{1,1},X_{1,2},…,X_{1,128},…,X_{128,128})$这样一个上万维的随机变量对图像上的每一个点的概率分布都进行估计的话，不论是对于计算量或是存储量都十分浪费。由于图像的像素点之间具有强相关性，因此往往只需要利用一个低维的随机向量，如$\textbf{Y} = (Y_{1},Y_{2},…,Y_{128})$就可以将图片的整体信息进行表述，就如PCA，SVD等算法，图片经过线性变换后，特征对角矩阵的元素事实上就可以基本涵盖图片的大部分信息。

假设图片的像素所代表的矩阵为可逆方阵，使用PCA对图片进行分解后: $X = U^Tdiag{y_1,y_2,…,y_{128}}U$。在此我们可以认为，$X$经过线性变换（左乘$U$矩阵，右乘$U^T$）后，变成了一个128维的对角方阵，这样的过程就可以当作是一种encoding（编码）的过程。而同样的对于这张图片，若需要进行秘密传输/压缩传输，只要我们知道$U$是多少，那么只需要对方给我们提供$Y$矩阵的128个对角元素，就可以由式子进行矩阵乘法，从而进行decoding（解码）。

我们假设存在一个$U$，对于海量的一类图片，他们经过上述的编码与解码过程，都能够实现$X$与$Y$的转换，那么这其中的对角线元素$y_1,y_2,…,y_{128}$的分布就可以由这海量的图片经过变换后得到的大量样本，进行概率分布的估计。估计完成后，即可对$Y$进行抽样，从而再经过$U$的decoding过程实现对于$X$的抽样。

遗憾的是，对于不同的$X$，其经过PCA以后，就会产生不同的$U$，因此无法对一类图片使用同样的U进行encoding与decoding，将X转换为128维的Y，这是由于PCA是一种线性变换，然而对于实际上的图片，像素与像素之间的关系往往都是非线性的，因此简单的线性编码与解码方法就不能对于一类图片进行处理。

那么从而我们假设$\textbf{$X_{N^2}$}$可以由一个n维的向量$\textbf{$Y_{n}$}$进行表示($n \ll N^2$)，但是$\textbf{X}$与$\textbf{Y}$之间存在一定的非线性函数关系：
\begin{align}
\textbf{Y} = f(\textbf{X})
\end{align}
由此，$f$函数就是一个encoder，实现了$X$到$Y$的变换；同样的：
\begin{align}
\textbf{X} = g(\textbf{Y})
\end{align}
$g$函数就是一个decoder，实现了$Y$到$X$的变换。
并且，若能够得到$Y$的概率分布函数$p(y)$，就可以从中进行抽样，得到$Y$，并且经过h函数$g$的变换，实现对于$X$的抽样。然而事实上，我们无法得知真实的$f$与$g$,因此在生成模型中，构造模型从而对$f$与$g$进行合理的逼近是至关重要的任务。

而利用深度学习中的神经网络，我们往往可以用“线性组合+非线性函数”的多层网络对于任意关系的函数进行逼近，即模型利用$f_\theta$ 对 $f$进行逼近，而我们要做的，实际上就是通过优化算法，针对特定的损失函数，对$f_\theta$ 与$g_\theta$中的参数$\theta$进行估计，从而得到一个接近真实$f$与$g$关系的函数逼近。此外，往往会假设$Y$服从正态分布，并且其元素相互独立。但是真正能够涵盖$X$信息的分布未必是$Y$所在的分布，我们称其为$Z$，$Z$能够表示$X$的近乎所有信息，却处在我们无法观察到的空间里，我们称$Z$为隐含变量(latent variable)。

论文概要：

本文的主要模型大体结构仍然是AutoEncoder的结构，受到NF(Normalization Flowing)模型的启发，在传统的AE中，加入双向函数层（可逆函数$\phi$），从而在encoding过程中，$X$经过$g$的变化变为$Y$以后，$Y$再经过$\phi$函数的变化，变为隐含变量$Z$；在生成过程中，由于$\phi$存在反函数，因此可以将$Z$经由$\phi^{-1}$的变换，转变为$Y$后，再利用$f$生成图像$X$。此外，出于传统AE训练方式效果较差以及直接进行端对端训练结果不收敛的现象，借鉴了GAN(Generative Adversarial Network)的对抗训练方式，对于$\phi^{-1}$以及$g$进行训练，(由于$\phi^{-1}$是$\phi$的反函数，所以在此$\phi$也训练完毕); 当$\phi^{-1}$与$g$训练完成后，再对$f$进行训练，其损失函数由上一步GAN的损失函数以及基于VGG（作为特征提取器）的损失函数两者共同构成。

前置内容：

VAE：

变分自编码器(VAE)是一种基于AE思想提出的模型。其思想利用$q(z)$这样一个可知的分布去代替逼近$p(z|X)$，利用KL散度对两者的差异进行衡量，在变分推导过程中，最终对于对数似然函数的变分下界进行优化，利用reparameterization Trick，将Z设定为标准正态分布后，从而对均值$\mu$与方差$\sigma$进行训练与估计。其理论十分复杂，这里就不详述，有兴趣可以点击这里进行学习以及更为详细的了解。虽然VAE的效果不如GAN出色，但是模型的背后具有一定的数学支撑，目前也作为主流的生成模型被广泛进行研究与使用。

GAN:

生成对抗网络(GAN)模型也是目前一类最火的生成模型之一。与传统AE不同的是，GAN省去了encoding的过程，直接从decoding出发，由隐含变量$Z$出发，通过$g$直接生成$S$。 并且，GAN中的“对抗”的意思是：由另一个神经网络与生成网络进行对抗学习，从而对于两个网络的参数进行学习。另一个网络$c$的目的是将生成器所生成的图像$X’$，与真实的训练图像$X$进行区分，其结构如下：

其优化函数如下：
\begin{align}
obj &= \min_{g} \max_{c} \mathbb{E}_{X \sim P(x)} [log(c(X))] + \mathbb{E}_{X’ \sim P(x’)} [log(1-c(X’))]\\
&= \min_{g} \max_{c} \mathbb{E}_{X \sim P(x) }[log(c(X))] + \mathbb{E}_{Z \sim P(z)} [log(1-D(g(Z)))]
\end{align}
(这里的数学期望可以直接用均值进行估计，也可以用蒙特卡洛方法对生成数据与真实数据进行抽样后进行估计)。

这个优化函数的意思是，优化$c$使得c对于生成的$X’$与真实的$X$区分能力最强，并且再优化$g$，使得在该条件下的$c$对于上述两种数据的区分能力最弱。这就是训练中$c$和$g$的对抗过程。在训练时，其实际迭代算法如下:
Algorithm -Optimization of GAN-:
循环T次：
step ①: 利用$G$生成$K$张$X’$；
step ②: 计算损失函数，计算当前损失函数对于$c$中参数的梯度；
step ③: 更新$c$中的参数，使得损失函数最大；
step ④: 计算损失函数，计算当前损失函数对于$g$中参数的梯度（只需计算第二项，第一项g未出现）；
step ⑤: 更新$g$中的参数，使得损失函数最小；
循环结束。

相比于VAE，GAN背后的数学原理相对容易理解，其网络结构的设计以及训练流程都突出了“对抗”这一概念，并且再诸多任务的处理上，生成的图像质量较高，因此目前也是重要的生成模型之一。更为详尽的学习，可以点击这里进行了解。

NF：(这一块的基础知识是本文的重点。)

标准化流(NF)基于概率论中，分布函数的转换这一简单的公式：
\begin{align}
p_{X}(x) = p_{Y}(y) |\frac{dy}{dx}|
\end{align}
$X \sim P_{X} ; Y \sim P_{Y}$
在高维分布的情况下，我们假设$Y = \phi(X)$,以上公式转换为:
\begin{align}
p_{X}(x) = p_{Y}(y) |det(J_{yx})|
\end{align}
其中，$J_{yx}$是y关于x的雅克比矩阵, 其元素为$J_{ij} = \frac{\partial y_i}{ \partial x_j}$。
对于标准化流，我们在此要求$\phi: X \rightarrow Y$是双射， 即 $\phi$具有反函数$\phi^{-1}$，$\phi^{-1}: Y \rightarrow X$。
那么根据我们已经设定的$Y$的分布，就可以由$\phi^-1$关于$y$的雅克比矩阵从而对于$P_X(x)$的分布进行表示。这里着重介绍NF框架下的一种实现方式：NICE,也是本文中对于$Y\rightarraw Z \rightarrow Y$的实现方法。

NICE:

NICE(非线性独立组分估计)中，其优化目标是对于$X$的分布$p_X$进行极大似然估计，等价于:
\begin{align}
log(p_{X}(x)) = log(p_Y(\phi(x))) + log(| det(\frac{\partial \phi(x)}{\partial x}) |)
\end{align}
在$Y$的独立假设下，
\begin{align}
log(p_{X}(x)) = \sum_{d=1}^{D}log(p_{Y_d}(\phi_d(x))) + log(| det(\frac{\partial \phi(x)}{\partial x}) |)
\end{align}
然而，事实上，对于第二项的行列式，计算代价很大，NICE的提出了一种简单的网络构造方式，使得:
① $\phi$具有简单好算的反函数；
② $\phi$关于$x$的雅克比矩阵尽量稀疏，尽可能是上下三角阵，从而行列式的计算容易。
